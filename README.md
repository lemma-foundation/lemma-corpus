# lemma-corpus retired

`lemma-corpus` is retired. The public data repository for Lemma is now the
**Proof Atlas**: [`lemma-foundation/lemma-proof-atlas`](https://github.com/lemma-foundation/lemma-proof-atlas).

The Proof Atlas is the open, replayable record of validator-accepted Lean
proofs, plus the real Mathlib missing-proof task registries and snapshots. It
supersedes this repo.

```bash
git clone https://github.com/lemma-foundation/lemma-proof-atlas.git
```

This repository is kept only as a tombstone so old links have somewhere clear
to land. Do not publish new corpus data here.
